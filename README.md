# tp_final_algoritmos_2021
Sistema de Gestión para Constructora (Pascal)
Descripción: Proyecto desarrollado en 2021 para la materia Algoritmos y Estructuras de Datos. Es un sistema de gestión de consola que permite administrar empresas, proyectos de construcción, ventas y clientes.

Tecnologías:

Pascal (Compatible con Free Pascal / Turbo Pascal)

Manejo de archivos binarios (File of Record)

Funcionalidades destacadas:

ABM Completo: Altas, Bajas y Modificaciones de 5 entidades distintas.

Validaciones: Control de duplicados e integridad de datos.

Algoritmos: Implementación manual de ordenamiento de burbuja y corte de control para estadísticas.
